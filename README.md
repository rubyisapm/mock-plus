mock-plus
===========

mock-plus是mockjs的封装，实现更便捷的前端测试数据生成。

> * toDoc(input, output)<br/>
生成数据说明文档<br/>
input: json文件地址<br/>
output: 说明文档的保存地址<br/>

> * toMock(input, output)<br/>
生成mock数据模板<br/>
input: json文件地址<br/>
output: 数据模板的保存地址<br/>

> * generator(input, output)<br/>
生成mock数据<br/>
input: json文件地址<br/>
output: 测试数据的保存地址<br/>
